# macro_deck

## Ever loved the stream deck but ur wallet ran off? macro_deck is here for you!!!

This app transforms an Android smartphone into a high-speed, customizable virtual macro station for PC users. It solves the issue of expensive and inflexible physical macro hardware. It is designed for streamers, developers and gamers who need quick access to complex PC commands. This tool is useful because it provides a software-defined, budget-friendly alternative to premium hardware.

The app is a standalone mobile application developed using Flutter. It interacts with a PC-side Python background service via a local network connection rather than relying on external cloud APIs like Firebase.
